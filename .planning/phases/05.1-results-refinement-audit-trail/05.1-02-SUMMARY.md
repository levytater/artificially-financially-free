# Phase 05.1 Plan 02: Audit Breakdown Table Summary

**One-liner:** Year-by-year calculation audit table accessible as Math Breakdown tab, showing home equity growth, cost comparison, and net worth progression with full transparency.

## What Was Built

Created a detailed audit breakdown table component that displays all intermediate calculation values year-by-year, addressing the "show your work" requirement for advanced users and skeptics. The table is organized into three focused sections to avoid overwhelming column counts:

1. **Home Equity Growth** - tracks property value appreciation, mortgage paydown, and equity accumulation
2. **Cost Comparison** - compares annual and cumulative costs for buyer vs renter with monthly savings
3. **Net Worth Progression** - shows portfolio vs equity over time with color-coded difference

The audit table is accessible as a 4th tab ("Math Breakdown") in the chart area, alongside the existing Net Worth, Cash Flow, and Investment Growth placeholder tabs.

## Technical Implementation

**Audit table data sources:**
- Home equity metrics: `HousingProjection.yearlyProjection[i]` (homeValue, remainingBalance, homeEquity)
- Cost metrics: `YearlyComparison[i]` (annualRent, monthlySavings, buyerAnnualCost)
- Net worth metrics: `PortfolioYear[i].endBalance` and `YearlyComparison[i].buyerNetWorthWithSelling`

**Derived columns computed inline:**
- Annual appreciation: `yearData.homeValue - prevHomeValue`
- Cumulative rent: Running sum of `yearlyComparison[j].annualRent` for j=0 to i
- Buyer cumulative cost: Already available as `yearData.cumulativeCost`
- Net worth difference: `renterNetWorth - buyerNetWorthWithSelling` (color-coded)

**Layout pattern:**
- Each section has its own table with focused columns (5, 6, and 4 columns respectively)
- Independent horizontal scrolling per section via `overflow-x-auto`
- Alternating row backgrounds for readability (`even:bg-muted/10`)
- Year 0 initial position row for context
