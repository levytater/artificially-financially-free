  <action>
1. Create `src/components/theme-provider.tsx` -- a 'use client' wrapper around next-themes' ThemeProvider:
   ```typescript
   'use client'
   import * as React from 'react'
   import { ThemeProvider as NextThemesProvider } from 'next-themes'
   export function ThemeProvider({ children, ...props }: React.ComponentProps<typeof NextThemesProvider>) {
     return <NextThemesProvider {...props}>{children}</NextThemesProvider>
   }
   ```

2. Replace `src/app/globals.css` with the Phantom-inspired dark theme from the RESEARCH.md. Key requirements:
   - Import `tailwindcss`, `tw-animate-css`, and `shadcn/tailwind.css`
   - `@custom-variant dark (&:is(.dark *));`
   - `@theme inline {}` block mapping all shadcn color tokens to CSS custom properties
   - `:root {}` block with OKLCH values for the Phantom-inspired dark palette:
     - Background: deep dark (~`oklch(0.145 0 0)`)
     - Primary: purple (~`oklch(0.746 0.113 292)`) matching Phantom's `#ab9ff2`
     - Card/popover: slightly lighter dark (~`oklch(0.185 0 0)`)
     - Border: subtle white opacity (~`oklch(1 0 0 / 10%)`)
     - Sidebar colors with purple accents
     - 5 chart colors for future visualizations
   - `@layer base` block with `border-border` and `bg-background text-foreground`
   - All values use OKLCH format (not HSL) per shadcn/ui v4 standard
   - The `--font-sans` variable maps to `var(--font-inter)` for the Inter font

3. Replace `src/app/layout.tsx` with the root layout that sets up the full provider hierarchy:
   - Import Inter font from `next/font/google` as a variable font with `--font-sans` CSS variable

   - HTML tag: `lang="en"` and `suppressHydrationWarning` (required by next-themes)
   - Body: apply Inter font variable class + `font-sans antialiased`
   - Provider nesting order (outside to inside): ThemeProvider (attribute="class", defaultTheme="dark", disableTransitionOnChange) -> NuqsAdapter (from 'nuqs/adapters/next/app') -> {children}
   - Note: CalculatorProvider is NOT added here yet -- it will be added in Plan 02 after the context is created
