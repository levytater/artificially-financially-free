Create the following files:

**src/types/housing.ts** -- All interfaces for the housing cost engine:
- `ProvinceCode` type: union of 'AB' | 'BC' | 'MB' | 'NB' | 'NL' | 'NS' | 'ON' | 'PE' | 'QC' | 'SK'
- `TaxBracket` interface: `{ readonly from: number; readonly to: number; readonly rate: number }` (to = Infinity for last bracket)
- `FthbRebateConfig` interface: province-specific first-time buyer rebate config with maxRebate, fullExemptionUpTo, and optional partialExemptionUpTo
- `ProvinceLttConfig` interface: brackets array + fthbRebate (null if no rebate)
- `CmhcTier` interface: `{ readonly maxLtv: number; readonly rate: number }`
- `CmhcResult` interface: `{ premium: Decimal; pst: Decimal; totalMortgageAddition: Decimal }`
- `ClosingCostItem` interface: `{ name: string; defaultAmount: number; description: string }`
- `BuyingCosts` interface: result of buying cost calculation (ltt, legal, inspection, titleInsurance, cmhcPst, appraisal, total)
- `SellingCosts` interface: result of selling cost calculation (realtorCommission, legal, mortgageDischarge, total)
- `MonthlyPaymentBreakdown` interface: month, payment, principal, interest, remainingBalance (all Decimal)
- `YearlyHousingCost` interface: year, mortgagePayment, principalPaid, interestPaid, remainingBalance, propertyTax, maintenance, homeInsurance, homeValue, homeEquity, totalAnnualCost, cumulativeCost (all Decimal except year)
- `UpfrontCosts` interface: downPayment, cmhcPremium, cmhcPst, ltt, buyingClosingCosts, totalCashRequired (all Decimal)
- `ExitPosition` interface: homeValue, remainingMortgage, sellingCosts, netProceeds (all Decimal)
- `HousingProjection` interface: upfrontCosts, yearlyProjection array, exitPosition
- `RentProjection` interface: year, monthlyRent, annualRent (Decimal), and rentProjection array

Import Decimal from `@/lib/decimal`. Use `import type` where possible.

**src/lib/data/provinces.ts** -- Province codes and defaults:
- `PROVINCE_NAMES` record mapping ProvinceCode to full province name
- `PROVINCE_CODES` array of all 10 ProvinceCode values
- `DEFAULT_PROPERTY_TAX_RATES` record mapping ProvinceCode to default property tax rate as percentage (reasonable province-level approximations: ON 1.0%, BC 0.5%, AB 0.8%, QC 1.1%, MB 1.5%, SK 1.3%, NB 1.3%, NS 1.3%, PE 1.0%, NL 0.9%)

**src/lib/data/ltt-brackets.ts** -- LTT brackets for ALL 10 provinces:
Use the exact bracket data from 02-RESEARCH.md Province Data Reference section. Store as `LTT_CONFIG: Record<ProvinceCode, ProvinceLttConfig>`. Include FTHB rebate configs for ON (max $4,000, full exemption up to $368,333), BC (full exemption up to $835K, partial to $860K, max $8,000), and PE (full exemption). For AB and SK which have no LTT (just registration fees), use a single bracket with 0% rate and store the fee calculation note in a comment. For NL, model the registration fee structure ($100 base + $0.40 per $100 over $500). For NS, use flat 1.5% (Halifax default). For NB, use flat 1%.

**src/lib/data/cmhc-rates.ts** -- CMHC premium table:
- `CMHC_TIERS` array with 6 tiers per research: maxLtv 0.65/0.75/0.80/0.85/0.90/0.95 with rates 0.60%/1.70%/2.40%/2.80%/3.10%/4.00%
- `CMHC_AMORTIZATION_SURCHARGE` = 0.0020
- `CMHC_PST_RATES` record: ON 0.08, QC 0.09, SK 0.06
- `CMHC_MAX_PURCHASE_PRICE` = 1499999 (as of Dec 2024)

**src/lib/data/closing-cost-defaults.ts** -- Default line items:
- `BUYING_COST_DEFAULTS`: legal $2000, inspection $500, titleInsurance $300, appraisal $0
- `SELLING_COST_DEFAULTS`: realtorCommissionRate 0.05, legal $1000, mortgageDischarge $300

- `DEFAULT_APPRECIATION_RATE` = 0.03 (3% nominal)

All data files use `as const` where applicable for type narrowing.

  <action>
Create 4 calculation functions and a test file:

**src/lib/calculations/property-tax.ts:**
- `calculatePropertyTax(homeValue: Decimal, taxRatePercent: Decimal): Decimal` -- Returns annual property tax. Formula: `homeValue * taxRatePercent / 100`.
- `calculateMaintenance(homeValue: Decimal, maintenanceRatePercent: Decimal): Decimal` -- Returns annual maintenance cost. Formula: `homeValue * maintenanceRatePercent / 100`.

**src/lib/calculations/appreciation.ts:**
- `calculateAppreciatedValue(initialValue: Decimal, annualRatePercent: Decimal, years: number): Decimal` -- Returns home value after N years of compound appreciation. Formula: `initialValue * (1 + rate/100)^years`.
- `calculateAppreciationSchedule(initialValue: Decimal, annualRatePercent: Decimal, totalYears: number): Decimal[]` -- Returns array of values for years 0 through totalYears.

**src/lib/calculations/rent.ts:**
- `calculateRentForYear(initialMonthlyRent: Decimal, annualIncreasePercent: Decimal, year: number): Decimal` -- Returns monthly rent for a given year (year 0 = initial). Formula: `initialRent * (1 + rate/100)^year`.
- `calculateAnnualRent(monthlyRent: Decimal): Decimal` -- Returns `monthlyRent * 12`.
- `calculateRentSchedule(initialMonthlyRent: Decimal, annualIncreasePercent: Decimal, totalYears: number): Array<{ year: number; monthlyRent: Decimal; annualRent: Decimal }>` -- Returns full rent projection.

**src/lib/calculations/closing-costs.ts:**
- `calculateBuyingCosts(inputs: { legalFees?: number; homeInspection?: number; titleInsurance?: number; appraisalFee?: number; ltt: Decimal; cmhcPst: Decimal }): BuyingCosts` -- Calculates total buying closing costs. Uses defaults from `closing-cost-defaults.ts` when optional params not provided. LTT and CMHC PST come from other calculation modules (passed as params since they're calculated elsewhere).
- `calculateSellingCosts(salePrice: Decimal, inputs?: { commissionRate?: number; legalFees?: number; mortgageDischarge?: number }): SellingCosts` -- Calculates total selling costs. Default commission 5%, legal $1000, discharge $300.

ALL functions use Decimal.js exclusively -- no native JS `+`, `-`, `*`, `/` on financial values. Import `{ Decimal }` from `@/lib/decimal`.
