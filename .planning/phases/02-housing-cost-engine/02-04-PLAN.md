`calculateHousingProjection(input: HousingProjectionInput): HousingProjection`

Implementation flow:
1. Convert all numeric inputs to Decimal
2. Calculate CMHC premium and PST
3. Determine actual mortgage principal: `purchasePrice - downPayment + cmhcPremium`
4. Calculate LTT (with FTHB rebate if applicable)
5. Calculate buying closing costs (passing LTT and CMHC PST)
6. Build UpfrontCosts: downPayment, cmhcPremium, cmhcPst, ltt, buyingClosingCosts, totalCashRequired
7. Generate amortization schedule for the full amortization period
8. Loop year 1 through timeHorizon:
   - Calculate appreciated home value for this year
   - Sum mortgage payments for months in this year (from amortization schedule; $0 if amortization complete)
   - Sum principal and interest portions
   - Get remaining balance at end of year
   - Calculate property tax based on home value (use start-of-year value or appreciated value -- be consistent)
   - Calculate maintenance based on home value
   - Add home insurance
   - Calculate homeEquity = homeValue - remainingBalance
   - Calculate totalAnnualCost = all cash outflows
   - Track cumulativeCost
9. Calculate exit position:
   - homeValue at end of timeHorizon
   - remainingMortgage from amortization schedule (or $0 if paid off)
   - sellingCosts via calculateSellingCosts
   - netProceeds = homeValue - remainingMortgage - sellingCosts.total

Use defaults from `closing-cost-defaults.ts` for optional parameters (propertyTaxRate, maintenanceRate, homeInsurance, appreciationRate).
