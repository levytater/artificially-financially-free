**GREEN phase — implement:**

Create `src/lib/calculations/comparison.ts`:

Import from Phase 2 and Phase 3 modules:
- `calculateHousingProjection` + `HousingProjectionInput` from `housing-projection`
- `calculateRentSchedule` from `rent`
- `calculateSellingCosts` from `closing-costs`
- `calculateCombinedMarginalRate` from `income-tax`
- `calculateAfterTaxReturn`, `calculatePortfolioGrowth` from `investment`
- `ComparisonInput`, `ComparisonResult`, `YearlyComparison` from `@/types/investment`
- `Decimal` from `@/lib/decimal`

Implement `findBreakEvenYear(yearlyComparison: YearlyComparison[], mode: 'withSelling' | 'withoutSelling'): number | 'never'`:
- Iterate yearlyComparison, find first year where buyer net worth (in specified mode) >= renter net worth
- Return year number, or 'never' if not found

Implement `calculateRentVsBuyComparison(input: ComparisonInput): ComparisonResult`:

Step-by-step composition:
1. **Housing projection:** Build `HousingProjectionInput` from `ComparisonInput` fields, call `calculateHousingProjection()`. This gives upfront costs, year-by-year buyer costs, and exit position.

2. **Marginal tax rate:** If `taxRateOverride` is provided (not null/undefined), use it directly as a Decimal. Otherwise, call `calculateCombinedMarginalRate(input.annualIncome, input.province)`.

3. **After-tax return:** Call `calculateAfterTaxReturn(input.investmentReturnPercent, marginalTaxRate)` — where marginalTaxRate is the percentage (e.g., 29.65). Returns annual after-tax decimal rate (e.g., 0.0596).

4. **Lump sum:** Renter invests buyer's cash outlay on day 1 = downPayment + total buying closing costs (from `housingProjection.upfrontCosts.totalCashRequired`). Per CONTEXT.md: "Down payment + closing costs the buyer would have spent".

5. **Monthly savings per year:** For each year 1..timeHorizon:
   - Buyer monthly cost = `yearlyHousingCost.totalAnnualCost / 12`
   - Renter monthly rent = from rent schedule (year index from `calculateRentSchedule` — use year N's monthlyRent for month-by-month within year N; since rent schedule returns year 0..N, use index equal to the comparison year)
   - Monthly savings = `max(0, buyerMonthlyCost - renterMonthlyRent)` (clamped per CONTEXT.md)
   - Build `Decimal[]` array of monthly savings per year for portfolio growth input

6. **Portfolio growth:** Call `calculatePortfolioGrowth(lumpSum, yearlyMonthlySavings, afterTaxReturn)`

7. **Year-by-year comparison:** For each year 1..timeHorizon, build `YearlyComparison`:
   - `renterNetWorth` = portfolio endBalance at that year
   - `buyerNetWorthWithoutSelling` = homeValue - remainingMortgage (from housing projection)
   - `buyerNetWorthWithSelling` = homeValue - remainingMortgage - sellingCosts at that year's home value. Call `calculateSellingCosts(homeValueAtYear)` for each year to get selling costs at that year's appreciated value.
   - `monthlySavings` = the monthly savings amount for that year
   - `annualRent` = from rent schedule
   - `buyerAnnualCost` = from housing projection yearly data

8. **Break-even:** Call `findBreakEvenYear` twice — once with 'withSelling', once with 'withoutSelling'.

9. **Return** `ComparisonResult` with all assembled data.

Run tests, ensure all pass. Commit RED then GREEN separately per TDD convention.
